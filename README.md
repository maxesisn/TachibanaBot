##使用方法
1.clone本项目
2.执行
```
git submodule update --init --recursive
```
下载子模块文件
3.按照[原项目](https://github.com/Ice-Cirno/HoshinoBot)的说明，补全缺失的文件

##更新方法
使用
```
git pull
```
命令更新项目本体，如有修改请自行merge

使用
```
git submodule update --remote
```
命令更新子模块
